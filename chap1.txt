머신러닝이랑 ?
기계에게 학습시키는거
기계는 사진을 보고 알고리즘을 통해 인식을 한다
이런 ㅇ라고리즘을 기계가 스스로 하게 만드는것 == 머신러닝

ex)스팸메일 필터링
머신러닝 X 인간이 직접 제목으로 필터링
머신러닝 O 컴퓨터가 스스로 단어 선정 후 필터링

머신러닝의 종류
1. Supervised Learning : 데이터에 정답이 있고 정답예측 모델 만들 때
2. Unsupervised Learning : 데이터에 정답이 없어 컴퓨터가 스스로 비슷한것으로 묶는다. ex) 추천알고리즘
3. Reinforcement Learning : 게임을 가르치는것과 같은원리 | 성공시 +1, 실패시 -10을 적용하여 Score를 가장 높게 받을수 있는 방법을 컴퓨터가 판단

수능성적 예측
6월 60
9월 80
수능 

ex) 6월 x0.5 + 9월 x0.5 = 70
= 6월 x w1 + 9월 x w2 + b = Y
컴퓨터에게 w1, w2를 예측하게 하는것 <- 머신러닝
여기서 w1, w2 == weight(가중치) 결과값에 얼마나 영향을 주는가
b = bias (결과값에 영향을 주는 값 (데이터를 제외하고))
크
W 오차를 최소화! 데이터셋을 이용하여 weight 값 수정
