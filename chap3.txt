손실함수로 컴퓨터에게 오차를 알려줘야함

중간 레이어 노드에 연결된 모든 이전 노드들을 가중치를 곱해서 더한 값을 저장한다.
처음엔 가중치를 모르기 때문에 미지수 w를 사용한다.

중간레이어 -> 히든 레이어 (h)로 표현

ex)
h1 = 50*0.2 + 60^0.1 = 16
h2 = 50*0.3 + 60*0.2 = 27
return = h1*0.5 + h2*0.4 = 18.8

오차를 계산하는법 !

실제 데이터를 이용하여 비교한다.
실제 결과값에서 예측한 값을 빼준것에 절댓값을 씌운 뒤 평균을 낸다. or 제곱 후 평균을 낸다(평균 제곱 오차). <- 손실함수(loss function)
이러한 오차값의 평균을 최소화 하는 w값을 찾는다.
손실함수 :  1/n Sig(^y - y)^2 (정수예측)
            -1/n Sig[y*log(^y) + (1-y)*loh(1-^y)] (확률예측)
